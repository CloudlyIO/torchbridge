## Quick Install

```bash
# Clone and install
git clone https://github.com/KernelPyTorch/kernel-pytorch.git
cd kernel-pytorch
pip install -e .

# Verify
python -c "import kernel_pytorch; print(f'v{kernel_pytorch.__version__} ready!')"
```
